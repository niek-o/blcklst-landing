## 🔨 Getting Started

Below are the instructions for running the app for development usage.

### 🏡 Running locally

1. Install dependencies with yarn using:

   ````shell
   $ yarn install
   ````

2. Run development server using:
   ````shell
   $ yarn dev
   ````

### 🏗️ Building the app

1. Install dependencies with yarn using:

   ````shell
   $ yarn install
   ````

2. Build app using:
   ````shell
   $ yarn build
   ````
